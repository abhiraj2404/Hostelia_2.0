# React + TypeScript + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

## How to add Shadcn Components

To add a shadcn component to the project, use the following command:

```bash
npx shadcn@latest add <component-name>
```

## How to setup Routing of pages

For routing of pages please refer to the React Router docs. Link - https://reactrouter.com/start/declarative/routing

## How to do State Management (Redux Toolkit)

Note that we are using `Redux-toolkit` and not base `Redux` our project.

For any doubts refer to the Redux toolkit docs. Link - https://redux-toolkit.js.org/tutorials/quick-start

To understand and get started with redux you can also watch a youtube tutorial by Hitesh. Link - https://www.youtube.com/watch?v=1i04-A7kfFI
